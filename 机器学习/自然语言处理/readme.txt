运行main.py即可得到：
一、文本数据的基本处理：1.JSON转CSV。2.读数据。3.删除无效列，删除重复行。4.取出content。5.正则表达式取出数字和英文字母。6.去掉空行。
二、文本数据的高级处理：1.载入自定义词典。2.分词。3.载入停用词。4.去停用词。5.去空格。6.生成词云图。
三、文本数据分析：1.词频统计。2.词性标注。3.名词提取。4.TF-IDF。5.关键字提取。
四、主题提取：1.LDA模块实现。
五、情感分析：1.baidu-aip实现。可得到“分类置信度”，“情感极性分类结果0负向1中性2正向”，“属于积极类别概率”，“属于消极类别概率”。
六、K-means分类：1.K-means聚类实现。
